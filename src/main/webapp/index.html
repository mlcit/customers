Customers

POST /api/v1/customers - create customer

  • FormParam firstName - set firstName to param
  • FormParam lastName - set lastName to param
  • FormParam birth - set birth to param [yyyy-MM-dd]

PUT /api/v1/customers/{id} - update customer with id

  • FormParam firstName - set firstName to param
  • FormParam lastName - set lastName to param
  • FormParam birth - set birth to param [yyyy-MM-dd]

GET /api/v1/customers/{id} - get customer with id

GET /api/v1/customers - list customers by parameters

  • QueryParam firstName - customers whose firstName starts with param
  • QueryParam lastName - customers whose lastName starts with param
  • QueryParam birthFrom - customers who were born after param (incl.) [yyyy-MM-dd]
  • QueryParam birthTo - customers who were born before param (incl.) [yyyy-MM-dd]

DELETE /api/v1/customers/{id} - delete customer with id

DELETE /api/v1/customers - delete customers by parameters

  • QueryParam firstName - customers whose firstName starts with param
  • QueryParam lastName - customers whose lastName starts with param
  • QueryParam birthFrom - customers who were born after param (incl.) [yyyy-MM-dd]
  • QueryParam birthTo - customers who were born before param (incl.) [yyyy-MM-dd]
